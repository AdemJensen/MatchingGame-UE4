## 简介

进入游戏后，可看到主界面，单击“开始游戏”即可开始游玩。

游戏将会给予一个 4 \* 4 的方格矩阵，每个方块被按下后上方将会出现一个数字。连续按下两个数字后，若两个数字相同，则方块被消除。否则，方块上的数字消失。消除所有方块以取得游戏胜利。
